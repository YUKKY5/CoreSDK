Value

Gets or sets the value. The value is stored in the native interaction property for which this is a wrapper. The value must be compatible with one of the types specified by PropertyValueType. In case the value to assign the interaction property to is not compatible with any of the PropertyValueTypes, but is a composite type, then a PropertyBag will automatically be created and populated with the values of the properties and fields of the composite type value.
